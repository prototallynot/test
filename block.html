About Russian Block Code

Russian block code (also known as pseudo-Cyrillic, faux Cyrillic, or "Renglish") is a method of writing English words using Cyrillic characters that look similar to Latin letters. This creates text that appears foreign at first glance but can be read by English speakers with some effort.

Example 2: Longer Text

Example 3: Reversal
